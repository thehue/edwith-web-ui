요약정리 
이미지 리스트는 float:left; 속성을 사용하여 블록요소를 좌측으로 이어붙게 만들었습니다.
loat을 사용할 경우 부모 요소가 자식 높이 값을 인지할 수 있도록 가상 선택자 :after에 clear 속성 적용하여 float을 해제하는 방법에 대해 알아보았습니다.

 말줄임 처리를 하기 위해 고정 너비 값을 입력한 후 text-overflow, overflow, white-space 속성을 사용하였습니다. 
 
 여러줄 말줄임의 경우 webkit 계열 브라우저와 ie에서 적용될 수 있도록 각 브라우저를 위한 코드에 대해서도 알아보았습니다.
마지막으로 액자 효과를 주기 위해 가상 선택자와 position을 이용하여 섬네일 위로 외곽선을 노출하고, 
반투명한 배경을 적용을 위해 rgba 이용한 컬러와 투명도 적용해 보았습니다.
