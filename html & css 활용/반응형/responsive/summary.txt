CSS 코드의 우선순위

반응형 웹의 각 실습에서 강조한 내용 중에 모바일 CSS는  태블릿의 CSS 밑에 작성이 되어야 하고, 
태블릿의 CSS는 PC의 CSS 밑에 존재해야 한다는 내용입니다. 이는 CSS의 선택자 우선순위에 관련된 내용입니다.

CSS의 코드는 같은 소스라면 먼저 작성된 소스보다 나중의 소스가 덮여씌여지는 특성이 있습니다. 
이런 특성을 캐스캐이딩이라고 합니다. 이는 CSS를 작성하는데 있어서 아주 중요한 특성 중에 하나이며 기초강의에서 자세하게 다루고 있습니다. 

이번 실습에서 중요한 이유는 PC가 기준으로 태블릿의 환경일 때는 태블릿 코드가 더 우선이 되고 모바일에서는 모바일코드가 최우선이 되도록 작성해야 하기 때문입니다.
이것은 미디어쿼리의 규칙이 더 많을 수록 우선권을 갖기 때문에 모든 상황을 잘 이해하고 미디어쿼리를 적용해야 합니다.
